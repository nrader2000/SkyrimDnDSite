About Elder Scrolls TableTop

This is the amazing game series of the Elder Scrolls focuesed into a tabletop style game! I continue to be in constant development of this game! I try to improve the gameplay with the new combat system that I mixed in from other D&D style tabletop games!The combat in this game incorperates some of series' core mechanics, as well as some new stuff to make the game feel more like a tabletop RPG. The game is in constant development and I'm always looking for ways to improve the game so please contact me with any advise for what I could do to make this game better for everyone to enjoy!
